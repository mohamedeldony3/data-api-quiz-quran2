## نظرة عامة

مشروع "معلمة الدرر" يعتبر قاعدة بيانات JSON تحتوي على مجموعة من الاختبارات العلمية في مجالات متنوعة من العلوم الشرعية. يتيح هذا المشروع للمستخدمين تقييم وتحسين مستواهم في المعرفة الدينية من خلال الوصول إلى بيانات JSON الموثوقة.

## Overview

The "Muealimat-al-darar" project is a JSON database containing a set of scientific tests in various fields of Islamic sciences. This project allows users to assess and improve their level of religious knowledge by accessing reliable JSON data.

## البيانات 

#### main.json الملف الرئيسي

**Example**

```js
{
    "description": "اسئلة تستطيع من خلالها تقويم مستواك في العلوم الشرعية وتطوير حصيلتك العلمية في مجالات متنوعة. يحتوي على 3 اختبارات علمية تتفاوت في الصعوبة، وكل اختبار يتألف من 20 سؤالا، وجميع الإجابات مصدرها موقع الدرر السنية.",
    "categories": [
        {
            "id": 4,
            "arabicName": "الفقه",
            "englishName": "figh",
            "description": "يقول الرسول عليه الصلاة والسلام (من يرد الله به خيرا يفقه في الدين)، احرص على الخير والتفقه في الدين",
            "path": "/database/figh.json",
            "icons": "/database/icons/4.svg"
        },

        // المزيد من الفئات
    ]
}
```


- **description (وصف المشروع)**: String
- **categories (الفئات)**: Array
- - **id (معرف الفئة)**: Number
- - **arabicName (الاسم بالعربي)**: String
- - **englishName (الاسم بالإنجليزية)**: String
- - **description (الوصف)**: String
- - **path (المسار)**: String
- - **icons (الأيقونة)**: String


#### category || englishName.json - الملف الخاص بالفئة


**Example**

```js
{
  "id": 4,
  "category": "الفقه",
  "description": "يقول الرسول عليه الصلاة والسلام (من يرد الله به خيرا يفقه في الدين)، احرص على الخير والتفقه في الدين",
  "icons": "/database/icons/figh.svg",
  "DataArray": [
    {
      "englishName": "tahara",
      "arabicName": "الطهارة",
      "files": [
        {
          "level": 1,
          "filename": "level-1.json",
          "path": "/database/figh/tahara/level-1.json"
        },
        {
          "level": 2,
          "filename": "level-2.json",
          "path": "/database/figh/tahara/level-2.json"
        },
        {
          "level": 3,
          "filename": "level-3.json",
          "path": "/database/figh/tahara/level-3.json"
        }
      ]
    },
    // المزيد من المواضيع
  ]
}
```


- **id (معرف الفئة)**: Number
- **category (الفئة)**: String
- **description (الوصف)**: String
- **icons (الأيقونة)**: String
- **DataArray (مصفوفة البيانات)**: Array
- - **englishName (الاسم بالإنجليزية)**: String
- - **arabicName (الاسم بالعربي)**: String
- - **files (الملفات)**:
- - - **level (المستوى)**: Number
- - - **filename (اسم الملف)**: String
- - - **path (المسار)**: String



#### Question Files (level-1.json, level-2.json, level-3.json)


**Example**

```js
[
  {
    "id": 133,
    "q": "المُخاطَبُ في قولِه تعالى: {فَإِنْ حَاجُّوكَ فَقُلْ أَسْلَمْتُ وَجْهِيَ لِلَّهِ وَمَنِ اتَّبَعَنِ}:",
    "level": 1,
    "link": "https://dorar.net/tafseer/3/7",
    "section": "aali-imraan",
    "answers": [
      {
        "answer": "مُوسى عليه السلامُ",
        "t": 0
      },
      {
        "answer": "عِيسى عليه السلامُ",
        "t": 0
      },
      {
        "answer": "مُحمدٌ صلَّى اللهُ عليه وسلَّمَ",
        "t": 1
      }
    ]
  },
  // المزيد من الأسئلة
]
```


- **id (المعرف)**: Number
- **q (السؤال)**: String
- **level (المستوى)**: Number
- **link (الرابط)**: String
- **section (القسم)**: String
- **answers (الإجابات)**: Array
- - **answer (الإجابة)**: String
- - **t (نوع الإجابة)**: Number رقم 0 إجابة خاطئة رقم 1 إجابة صحيحة
